# 🏦 Bank Fraud Detection — Predictive Model

This project is used as a training case for a predictive model focused on fraud detection in bank transactions. It also serves as a practice exercise for data presentation, feature engineering, model selection, hyperparameter tuning, and the interpretation of results through data visualization.

---

## Objectives

- Detect fraudulent transactions using machine learning techniques
- Practice feature engineering and data preprocessing on real-world financial data
- Evaluate and compare multiple classification models
- Present findings through clear data visualizations

## 📊 Dataset

> Dataset source: *[Bank Fraud](https://www.kaggle.com/datasets/nafiulislam490/bank-transaction-fraud-detection-dataset)*

The dataset consists of a comprehensive collection of bank transaction records, including features such as transaction amount, account behavior, and historical patterns. It is designed for fraud detection research and machine learning practice.
